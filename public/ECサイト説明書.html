3
カートの仕組み（CartContext）

カートの中身は CartContext というReactの仕組みで管理しています。
これにより、どのページからでも同じカート情報にアクセスできます。

CartContext が持つ機能

addItem( ) → カートに追加
removeItem( ) → 削除
updateQuantity( ) → 数量変更
clearCart( ) → 全削除
totalCount → 合計個数
totalPrice → 合計金額

addItem でカートに追加する

カートページ

items updateQuantity removeItem を使用

💾 カートの中身は localStorage（ブラウザのメモリ） に自動保存されます。
ページをリロードしても、ブラウザを閉じて開き直しても消えません。
4
商品をカートに入れるまでの流れ
1

Supabaseから商品データを取得

supabase.from('products').select('*') を実行します。
データベースに商品がなければ、サンプルデータ（6件）を自動で表示します。

2

商品一覧ページに表示

カテゴリボタンで絞り込みができます。商品カードをクリックすると詳細ページへ移動します。

3

詳細ページで「カートに追加」をクリック

addItem(product) が呼ばれ、カートに追加されます。
ヘッダーのバッジがすぐに増えます。

4

カートページで確認・注文

数量の変更・削除・合計金額の確認ができます。
「注文を確定する」をクリックするとカートが空になり、完了画面が表示されます。

5
データベースの構造（Supabase）

products テーブル（商品データ）

列名 / 型 / 説明
id
uuid
商品の一意なID（自動生成）
name
text
商品名（例：クラシックTシャツ）
description
text
商品の説明文
price
integer
価格（円）例：2980
image_url
text
商品画像のURL（なくてもOK）
category
text
カテゴリ名（例：ウェア、キッチン）
stock
integer
在庫数（0のとき「在庫なし」と表示）

Supabaseへの登録方法

Supabase → SQL Editor → supabase/products.sql の内容を貼り付けて実行
