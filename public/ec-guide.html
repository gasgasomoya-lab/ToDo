SLIDE 04 / カート管理

CartContext — カート情報を全ページで共有する仕組み

CartContext が持つ機能
addItem( ) → カートに追加
removeItem( ) → 削除
updateQuantity( ) → 数量変更
clearCart( ) → 全削除
totalCount → 合計個数
totalPrice → 合計金額

💾 カートの中身は localStorage に自動保存されます。ページをリロードしても消えません。
SLIDE 05 / データフロー

商品をカートに入れるまでの流れ

1
Supabaseから商品データを取得
supabase.from('products').select('*') を実行。データがなければサンプルデータを使用。
2
商品一覧ページに表示
カテゴリで絞り込みできる。商品カードをクリックすると詳細ページへ。
3
詳細ページで「カートに追加」をクリック
addItem(product) が呼ばれ、CartContextのstateが更新される。ヘッダーのバッジが即座に増える。
4
カートページで確認・注文
数量の変更・削除ができる。「注文を確定する」でカートが空になり、完了画面が表示される。
